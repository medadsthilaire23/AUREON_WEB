# AUREON_WEB

AUREON_WEB es una aplicación web minimalista desarrollada en **Flask** que permite a los usuarios registrar y monitorear **ASINs de Amazon**.  
Está diseñada como un sistema modular, con capacidad de escalar en el futuro para agregar análisis, predicciones y módulos adicionales.

## Funcionalidades principales

- Registro de ASINs reales mediante un formulario web.
- Persistencia de datos usando un archivo `JSON` (`asin_data.json`).
- Visualización de todos los ASINs registrados con fecha y hora de registro.
- Arquitectura modular y escalable para integrar futuras funcionalidades de AUREON.
- Interfaz simple con plantillas HTML y archivos estáticos CSS/JS.
